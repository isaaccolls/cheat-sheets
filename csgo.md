## commands

- show fps: (in game) `cl_showfps 1`
- show fps and net: `net_graph 1`
- show mouse sensitivity: `sensitivity`
- set mouse sensitivity: `sensitivity [value]`
- off menu music: `snd_menumusic_volume 0`
